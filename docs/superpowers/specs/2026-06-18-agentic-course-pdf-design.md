# 《Production-grade Agentic Systems》中文合订本 PDF 设计

## 目标与范围

将 `docs/00-how-to-use-this-course.zh-CN.md` 至 `docs/22-designing-your-own-agent.zh-CN.md` 共 23 个 Markdown 文件合并为一份适合连续阅读、检索和打印的中文 PDF。源译文保持不变；最终文件写入 `output/pdf/production-grade-agentic-systems-zh-CN.pdf`。

## 内容结构

- 独立封面，明确中文书名、章节范围与生成日期。
- 目录覆盖 Ch.00-Ch.22，显示准确起始页码。
- 每章强制从新页开始，保留原有标题层级、段落、列表、引用、表格和分隔线。
- PDF 书签覆盖所有章节，便于阅读器侧栏导航。
- 不改写译文，不补充未经源文件支持的解释。

## 排版规则

- 使用 A4 页面和适合中文长文阅读的页边距、字号与行距。
- 正文使用可嵌入的中文字体；代码使用等宽字体。
- 页眉显示课程简称与当前章节，页脚显示连续页码。
- 标题不得孤立在页尾；表格、代码块和图形尽量避免跨页断裂。
- 代码块保留语言标识并使用克制的语法配色；长行优先换行，不能裁切。
- Mermaid 源代码渲染为矢量或高分辨率图形；失败时明确显示原始图代码，不能静默丢失。

## 生成架构

采用两阶段混合流程：先把 Markdown 转换为带统一样式的 HTML，渲染 Mermaid 图并通过 Chromium 输出正文 PDF；再读取正文的实际分页，生成带准确页码的封面和目录，合并为最终 PDF，并写入章节书签。

生成脚本与临时资源放在 `workspace/course-pdf/` 和 `tmp/pdfs/`。最终 PDF 只放在 `output/pdf/`。

## 校验标准

- 输入文件数量为 23，顺序严格为 Ch.00-Ch.22。
- PDF 可打开、页数合理，且能提取中文文本。
- 目录中的每个章节页码与正文起始页一致。
- 所有 40 个 Mermaid 块均被处理，所有代码块和表格均未丢失。
- 渲染全部页面为 PNG，自动检查空白页、页面尺寸和文本溢出风险，并人工抽检封面、目录、章节首页、代码密集页、表格页和图形页。
- 页面无乱码、黑块、裁切、重叠或不可读的小字。

## 非目标

- 不修改 `course/`、`references/` 或任何译文 Markdown。
- 不对译文做内容润色、增删或术语重写。
- 不制作 EPUB、MOBI 或单章 PDF。
